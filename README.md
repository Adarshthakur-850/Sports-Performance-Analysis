## 🧠 About the Project

**Sports Performance Analysis** is a fully functional analytics platform built to interpret and visualize athlete performance metrics. Leveraging modern data science libraries and interactive visuals, this project transforms raw sports data into insights that can benefit:

- Coaches tracking performance over time  
- Athletes identifying strengths & weaknesses  
- Analysts uncovering patterns and trends  

The application is designed as a **dashboard** that’s simple to use, effective for interactive exploration, and scalable for multiple sports.

## 🛠 Installation

Follow these steps to run the dashboard locally:

1. **Clone the repo**

```bash
git clone https://github.com/Adarshthakur-850/Sports-Performance-Analysis.git
cd Sports-Performance-Analysis
```

2. **Create a virtual environment**

```bash
python -m venv venv
```

3. **Activate it**

* On Windows:

```powershell
.\venv\Scripts\activate
```

* On macOS/Linux:

```bash
source venv/bin/activate
```

4. **Install dependencies**

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

1. Ensure you have Python 3.8+ installed
2. Activate your virtual environment
3. Run the application

```bash
python src/app.py
```

4. Open your browser at

```
http://localhost:8501   (for Streamlit)
or
http://127.0.0.1:8050   (for Dash)
```
